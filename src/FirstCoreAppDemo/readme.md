# 物料管理
## 假设需求
### 作为物料使用人员
1. 按照 _物料编码_ *Code* 检索物料信息
2. 浏览现有物料列表，包含列：
* 物料全名 *FullName*
* 物料名 *Name*
* 编码 *Code*
* 单位 *Unit*
3. 申请该物料

### 作为物料管理人员，在使用人员功能上增加
1. 添加一个物料信息
- 认为物料编码 存在层级定义过程，例如：
- 第一层编码类似 0100
- 则第二层类似   0100 0200
-   第三层类似   0100 0200 0345
2. 修改一个物料信息
* 修改一个物料编码时，只能修改本物料信息，涉及层次关系调整时，另行处理
3. 删除一个物料信息
4. 删除一个具有子分类或者子项的物料信息
